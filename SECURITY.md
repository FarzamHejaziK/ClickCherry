# Security Policy

## Supported Versions

Security support currently targets the latest `main` branch and the latest
published release.

## Reporting a Vulnerability

Security contact: `clickcherry.app@gmail.com`

For sensitive reports:

- Email `clickcherry.app@gmail.com` with details, impact, and repro steps.
- Do not include production secrets or private user data beyond what is required to reproduce.

For non-sensitive reports:

- Open a GitHub issue with label `security`.
